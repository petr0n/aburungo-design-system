# AburunGo Design Direction

## Short version

AburunGo brand direction

AburunGo should feel like a beautifully designed adult learning tool: calm, precise, and quietly confident. Think Muji notebook or Leica camera — craft-first, restrained, no forced personality in the product itself.

On top of that, the brand should have an editorial voice: considered, opinionated, and occasionally playful. Think magazine-like moments — sharp pull quotes, intelligent captions, satisfying progress language, subtle surprise. The playfulness should feel dry, warm, and earned, never cartoonish or bubbly.

The key tension is: serious object, lively mind.

Visual system: calm, minimal, structured, adult, tactile, trustworthy
Voice: concise, literate, slightly witty, encouraging without hype

## Long version

## 1. Core brand framing

AburunGo should feel like a **beautifully designed adult learning object with a human editorial voice**.

That means:

- **Product design** = calm, direct, precise, restrained
- **Brand voice / microcopy / campaigns** = intelligent, lightly playful, occasionally surprising
- **Character** comes from **judgment and writing**, not from illustration gimmicks or exaggerated UI

A useful rule:

> The interface should look more serious than the copy sounds.

This tension is the point.

---

## 2. Two layers of personality

To resolve the apparent contradiction, think of the brand as having two layers.

### Layer 1: System personality

This is the permanent foundation of the product.

Attributes:

- quiet competence
- adult
- tactile / crafted
- minimal but not sterile
- trustworthy
- focused
- elegant underuse of decoration

Reference qualities:

- Muji notebook
- Leica camera
- good editorial book design
- transit signage
- a language textbook designed by people with taste

### Layer 2: Editorial personality

This is where warmth enters.

Attributes:

- observant
- wry
- lightly playful
- motivating without cheerleading
- clever, not cute
- alive, not loud

Reference qualities:

- magazine pull quotes
- margin notes
- smart captions
- understated jokes
- satisfying progress language

Core rule:

> The core system stays calm. The editorial layer adds charm.

---

## 3. Brand spectrum

AburunGo should sit roughly here:

- **70%** calm / direct / sharp
- **20%** warm / human / encouraging
- **10%** playful / surprising

This balance matters.

### Too cold

Avoid drifting into:

- enterprise SaaS
- luxury-without-soul
- academic test-prep product
- blank minimalism
- overprofessionalism

### Too playful

Avoid drifting into:

- kawaii mascot app
- candy-colored gamification
- exaggerated emotional reactions
- cartoon Japanese clichés
- novelty-first edtech

---

## 4. Where playfulness belongs

Playfulness should not be treated as a full-system visual rule. It should appear in controlled, intentional places.

### Good places for playfulness

- lesson titles
- pull quotes
- progress celebrations
- empty states
- AI conversation prompts
- example sentences
- editorial callouts
- achievement naming
- occasional graphic moments

### Places that should stay clear and restrained

- navigation
- information architecture
- spacing system
- primary typography
- logo geometry
- settings and account flows
- drill interfaces where clarity matters

Rule:

> Play lives in accents, not structure.

---

## 5. Brand voice and writing guidance

### Voice qualities

- calm
- concise
- literate
- slightly witty
- encouraging without hype
- never childish
- never loud
- never anime-themed by default
- never faux-Zen
- never “bestie coach”

### Writing style

- short sentences
- clear verbs
- occasional dry humor
- small moments of delight
- enough restraint that playful lines feel special

### Emotional goals

Users should feel:

- “I trust this app.”
- “This app respects my intelligence.”
- “This is enjoyable to return to.”
- “Progress feels satisfying, not gamified.”

---

## 6. Visual design direction

Visual identity should be separated from expressive moments.

### Core visual identity

- minimal, structured layout
- strong typographic hierarchy
- restrained color system
- one or two warm accents, not a rainbow
- calm surfaces with occasional editorial contrast
- subtle tactility or paper-like texture if useful
- references from publishing, stationery, signage, and precision hardware

### Expressive layer

- quote-like cards
- highlighted phrases
- occasional oversized type
- interesting crop and composition
- restrained playful motion
- unexpected but tasteful sentence examples
- whitespace used so accent moments feel intentional

A useful shorthand:

> Design it like a premium field notebook for language learning, then let the editorial layer make it charming.

---

## 7. Logo direction

The logo should resolve the same tension as the product.

### Avoid

- mascot logo
- super-soft rounded kawaii wordmark
- literal rising sun / sakura / torii clichés
- hyper-tech startup symbol
- luxury serif mark that feels too fashion-only or expensive

### Prefer

- a wordmark-first identity
- typographic confidence
- strong legibility
- memorability through spacing, rhythm, cuts, or one subtle surprise
- a secondary symbol only if it grows naturally from the system

### Good symbol directions to explore

- annotation-inspired mark
- reading cue or editorial notation
- notebook stamp / seal logic
- bracket, cursor, emphasis, or study-mark logic
- something functional before decorative

A useful rule:

> The logo should feel serious at first glance and charming on second glance.

---

## 8. Do / don’t list

### Do

- premium but approachable
- adult learner, not child learner
- culturally aware without obvious Japanese tropes
- playful through editing and tone, not cartoon styling
- motivating through momentum and craft
- memorable through precision

### Don’t

- mascot-first
- bubbly edtech
- anime branding by default
- tourist-Japan iconography
- generic startup minimalism
- austere luxury
- exam-prep institutional tone

---

## 9. Creative brief paragraph

Use this as a compact summary when needed:

> AburunGo should feel like a beautifully designed adult learning tool: calm, precise, and quietly confident, like a Muji notebook or Leica object. On top of that foundation, it should have an editorial voice: considered, opinionated, and occasionally playful in small, well-placed moments. The product should never feel cartoonish, childish, or gamified, but it also should not feel sterile or overly professional. Playfulness should come through writing, pacing, example content, and select visual accents — not through mascots, loud colors, or novelty UI. The result should feel trustworthy, motivating, and enjoyable to return to.

---

## 10. North-star phrases

Use these to keep design explorations aligned:

- **AburunGo sits between “precision tool” and “editorial companion.”**
- **A serious object with a lively mind.**
- **Craft-first, wit-second.**
- **Quiet form, warm intelligence.**

If a concept does not support these phrases, it is probably drifting off-brief.

---

## Practical asks for a design agent

When exploring the brand, ask for:

1. **Three brand territories**
   - tool-first
   - editorial-first
   - balanced hybrid

2. **Three logo directions**
   - pure wordmark
   - wordmark + annotation symbol
   - wordmark + seal/stamp system

3. **A tone matrix**
   - calm ↔ playful
   - minimal ↔ expressive
   - academic ↔ conversational
   - premium ↔ approachable

4. **A UI expression map**
   Show where the brand can be expressive:
   - onboarding
   - lesson cards
   - streak/progress
   - AI chat
   - error states
   - empty states
   - review sessions

5. **A microcopy sample set**
   Write examples for:
   - onboarding headline
   - review complete message
   - streak message
   - wrong answer feedback
   - AI conversation prompt
   - empty state
   - one playful pull quote

This forces the voice to become concrete.

---

## Final recommendation

Bias the visual system toward **quiet competence**, and let the delight come from **editorial timing, language, and a few sharply chosen accent moments**.
